# Networking Outreach

Drafts personalized outreach messages for networking with target company
contacts, recruiters, and executive search firms.

## Status: Planned

This skill is stubbed for future development.

## Before You Start

1. Run `node scripts/validate-config.js` — if it exits non-zero, stop and show the error
2. Read `config/candidate.md` — candidate name, current role, target roles
3. Glob `references/writing-samples/*.md` — if any files exist, read them to
   calibrate tone before writing

## Intended Behavior

1. Accept target person/company and context (warm intro, cold outreach, recruiter)
2. Research the contact and company
3. Draft a concise, authentic message in Chris's voice
4. Tailor tone: peer-to-peer for VPs, respectful for recruiters, warm for referrals
5. Output message ready to send via LinkedIn or email

## Message Types

- Cold outreach to engineering leaders at target companies
- Warm introduction requests through mutual connections
- Executive recruiter relationship building
- Follow-up after networking events or conversations
